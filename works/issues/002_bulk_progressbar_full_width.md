**날짜**: 2025-11-12

**우선순위**: 중간
**관련 이슈**: -

---

## 개발자 요청

**배경**:
- 대량 전송 프로그레스바가 modal-footer 내부에 위치하여 너비가 작음
- 사용자가 전송 진행 상황을 명확히 보기 위해 모달 전체 너비로 표시 필요

**요청 사항**:
1. 프로그레스바를 모달 전체 너비만큼 길게 표시
2. 시각적 구분선 추가
3. 모달 하단에 딱 붙도록 배치

---

## 해결 방안

### 1. CSS 수정 전략
- `.progress-container`에 음수 margin 적용
- modal-footer의 padding을 상쇄하여 모달 전체 너비 차지
- 상단 구분선 추가

### 2. 기술적 접근
```css
.progress-container {
    margin: 20px -30px -20px -30px;
    padding: 15px 30px;
    background-color: #f8f9fa;
    border-radius: 0;
    border-top: 1px solid #dee2e6;
}
```
